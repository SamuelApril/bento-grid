# Bento Grid

This is my solution to the Bento Grid challenge from Frontend Mentor.

I built this project to practice CSS Grid, responsive layout, and arranging different cards inside one page section.

Built With

- HTML
- CSS
- CSS Grid
- Responsive design

What I Learned

This project helped me understand how a bento-style layout is structured and how different cards can work together inside one design.
The biggest learning point was understanding how grid-based layouts behave, especially when working with columns, rows, spacing, card sizing, and responsive changes.
I also became more comfortable debugging layout issues directly in the browser and testing changes step by step instead of randomly changing everything at once

Continued Development

I want to keep improving my understanding of:

- Responsive layouts
- Image positioning inside cards
- Better spacing decisions
- Cleaner HTML structure
- Building layouts that are easier to maintain later
